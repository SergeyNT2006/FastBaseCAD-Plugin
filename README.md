## 🎯 FB Command
FB [Enter] → P(Points)/L(Lines)/S(Styles)
Red line = height (h) ✓ Exact mm
Green line = projection (p) ✓
Layers: HEIGHTS/PROJECTIONS ✓

## 📥 Installation
Copy FastBaseCAD.bundle → C:\ProgramData\Autodesk\ApplicationPlugins
Restart AutoCAD → FB [Enter]
